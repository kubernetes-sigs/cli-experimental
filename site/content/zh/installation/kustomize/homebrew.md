description: >
    MacOS 系统可以使用 Homebrew 或 MacPorts 来安装 Kustomize。
---

[Homebrew](https://brew.sh) 用户可以：

```
brew install kustomize
```

[MacPorts](https://www.macports.org) 用户可以：

```
sudo port install kustomize
```
